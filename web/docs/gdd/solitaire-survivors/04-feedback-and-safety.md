# Feedback and Safety - 피드백 루프와 안전장치

## 개요

"Solitaire Survivors"는 솔리테어와 전투 시스템 간의 **양방향 피드백**을 통해 두 메카닉을 유기적으로 연결한다. 순방향(솔리테어→전투)은 플레이어의 카드 조작을 즉각적인 전투 이점으로 변환하고, 역방향(전투→솔리테어)은 전투 성과를 솔리테어 돌파구로 제공한다. 이를 통해 "잘 싸울수록 솔리테어가 쉬워지고, 솔리테어를 잘 풀수록 더 강해진다"는 상승 나선을 만든다.

동시에 Klondike 솔리테어의 본질적 막힘 문제(~25% 풀이 불가)에 대응하기 위해 4중 안전장치를 설계하여, 플레이어가 운에 의한 무기력감 없이 10분 세션을 완주할 수 있도록 보장한다.

---

## 순방향 피드백: 솔리테어 → 전투

플레이어의 솔리테어 조작이 전투에 미치는 영향. **액티브(의식적 추적)** 요소 2개, **패시브(자동 발동)** 요소 3개로 구성하여 인지 부하를 제한한다.

### 액티브 채널 (핵심 메카닉)

플레이어가 능동적으로 트리거하고 즉각 피드백을 받는 행위.

| 솔리테어 액션 | 전투 효과 | 시각/음향 피드백 | 전략적 의미 |
|-------------|----------|----------------|-----------|
| **Foundation 적재** | 투사체 Tier 상승 + 즉발 1발 + 넉백 | 카드 올릴 때 섬광 + 즉시 투사체 발사 + 적 경직 | 가장 강력한 전투 강화 수단. 언제 어떤 Suit를 올릴지가 핵심 전략 |
| **Tableau → Tableau 이동** | 체인 게이지 충전 | 게이지 바 증가 + 연속 이동 시 게이지 색상 변화 | 빠른 연속 이동으로 버스트 딜 준비 |

**Foundation 적재 상세**:
- **투사체 Tier 상승**: 해당 Suit의 Foundation 최고 Rank가 오를 때마다 투사체 수 재계산 (`ceil(Rank / 3)`, 최대 5발)
- **즉발 투사체**: 카드를 올리는 순간 화면 중앙에서 방사형으로 즉시 발사되는 강화 투사체 1발 (데미지 x1.5)
- **넉백 효과**: 즉발 투사체가 명중한 적은 0.3초간 경직 + 약간 뒤로 밀림
- **시각 강조**: Foundation에 카드가 올라가는 애니메이션 + 상단 전투 화면에서 해당 Suit 투사체가 짧게 빛남

**체인 게이지 시스템**:
- **충전 조건**: Tableau에서 Tableau로 카드(또는 카드 스택)를 성공적으로 이동할 때마다 게이지 +10%
- **가속 충전**: 10초 내 연속 이동 시 +15% (콤보 윈도우)
- **게이지 풀 (100%)**: "체인 브레이크" 발동
  - 효과: 3초간 모든 투사체 발사 속도 2배
  - 시각: 화면 테두리 황금빛 + 투사체 궤적에 잔상 효과
  - 음향: "체인!" 보이스 + 금속성 울림
- **게이지 감소**: 10초간 Tableau 이동이 없으면 초당 -5% 자연 감소
- **UI 표시**: 화면 좌측에 수직 게이지 바 상시 표시 (0~100%, 색상: 파랑→황금)

### 패시브 채널 (자동 보조)

플레이어가 일반적인 솔리테어 플레이 중 자동으로 얻는 보너스. 의식적 추적 불필요.

| 솔리테어 액션 | 전투 효과 | 게임 플레이 의미 |
|-------------|----------|---------------|
| **face-down 카드 뒤집기** | 소량 경험치 획득 (5 XP) | Tableau 정리 과정에서 자연스럽게 레벨업 진행 |
| **Stock 순환** (3장 드로우) | 0.5초 전체 적 감속 (이동속도 -30%) | 막힌 상황에서 Stock을 넘기는 행위가 방어적 가치 부여 |
| **빈 Tableau 열 생성** | 해당 영역에 "결계 존" 생성 | 빈 열을 만드는 전략적 선택에 전투 이점 추가 |

**결계 존 시스템**:
- **생성 위치**: 솔리테어 Tableau의 7개 열은 상단 전투 화면의 7개 세로 영역에 대응. 빈 열에 해당하는 전투 영역에 반투명 파란색 수직 라인 생성
- **효과**: 해당 영역에 진입한 적의 이동속도 50% 감소 (공격속도는 정상)
- **소멸 조건**: Tableau 열에 카드를 다시 놓으면 즉시 결계 소멸
- **전략적 가치**:
  - 최대 7개까지 가능하나 현실적으로 2-3개 유지 (솔리테어 플레이 공간 부족)
  - King이 필요한 빈 열을 만들기 위해 카드를 Foundation으로 밀어 올리는 선택에 전투 이점 부여
  - 적 진행 속도 조절 수단으로 활용 가능

## 역방향 피드백: 전투 → 솔리테어 (긍정적만)

전투 성과가 솔리테어에 미치는 영향. **부정적 역방향은 일체 배제**하여 하향 나선(negative spiral)을 방지한다.

### 설계 원칙: 긍정적 상승 나선

- **채택**: 전투를 잘할수록 솔리테어가 쉬워지는 보상
- **불채택**: 피격/보스 등장 시 솔리테어를 방해하는 패널티 (저주, 카드 조작 불가, Foundation 적재 딜레이 등)
- **근거**: "약해질수록 더 약해지는" 학습된 무력감(learned helplessness) 방지. 플레이어는 항상 "더 나아질 수 있다"는 느낌을 받아야 한다.

### 역방향 채널 3종

| 전투 이벤트 | 솔리테어 효과 | 발동 빈도 | 게임플레이 의미 |
|-----------|-------------|----------|---------------|
| **웨이브 클리어** | face-down 카드 1장 자동 reveal | 웨이브당 1회 (10회/판) | 생존 보상. 솔리테어 진행 가속 |
| **적 처치** | 15% 확률로 야생카드 드랍 | 적당량 (웨이브당 평균 5-8장) | 전투 성과를 솔리테어 돌파구로 전환 |
| **미니보스 처치** | 확정 야생카드 1장 | Wave 3, 6, 9 (3회/판) | 보스전 보상. 전략적 선택 기회 |

**웨이브 클리어 reveal**:
- 웨이브 종료 시 Tableau에 남아있는 face-down 카드 중 1장을 무작위로 자동 뒤집음
- face-down이 없으면 발동하지 않음
- 시각 효과: 뒤집힌 카드에 황금빛 반짝임

## 야생카드 시스템 (Wild Card)

적 처치 시 드랍되는 특수 카드 3종. 표준 52장 덱 외부의 "치트 카드"로 솔리테어 막힘을 돌파하는 핵심 안전장치.

### 야생카드 종류 (3종)

| 야생카드 | 드랍 확률 | 효과 | 듀얼 유즈 | 전략적 의미 |
|---------|----------|------|----------|-----------|
| **Joker (조커)** | 7% | 아무 Foundation에 올릴 수 있는 만능 카드. 해당 Foundation의 "다음 Rank"로 취급 | X | "어느 Suit에 쓸까" 선택. 막힌 Foundation 돌파 |
| **Peek (엿보기)** | 5% | face-down 카드 1장을 10초간 미리 볼 수 있음 (카드 뒷면에 반투명 오버레이로 표시) | X | 정보 제공. 숨겨진 카드 파악으로 전략 수립 |
| **Bomb (폭탄)** | 3% | **듀얼**: Foundation에 올리면 전체 화면 폭발 공격 / Tableau에 놓으면 face-down 카드 2장 자동 뒤집기 | O | "공격 vs 솔리테어 진행" 트레이드오프 |

**총 드랍 확률**: 적 처치 시 15% (7% + 5% + 3%)

### Joker 상세 규칙

- **사용 방법**: 야생카드 슬롯에서 드래그하여 원하는 Foundation에 드랍
- **Rank 취급 규칙**:
  - Foundation이 비어있으면 **Ace**로 취급 (게임 시작 조건)
  - Foundation에 이미 카드가 있으면 **현재 최상단 Rank + 1**로 취급
  - 예: Heart Foundation이 7까지 쌓인 상태 → Joker를 올리면 Heart 8로 취급
- **투사체 효과**: Joker를 올려도 Foundation 적재와 동일하게 즉발 투사체 + 넉백 + Tier 상승
- **시각**: Joker는 무지개 테두리, Foundation에 올려질 때 해당 Suit 색상으로 변환

### Peek 상세 규칙

- **사용 방법**: 야생카드 슬롯에서 Tableau의 face-down 카드를 클릭
- **효과**: 선택한 face-down 카드의 앞면이 10초간 반투명 오버레이로 보임 (다른 카드는 여전히 뒷면)
- **지속 시간**: 10초 후 오버레이 사라지고 다시 뒷면으로 (실제 카드 상태는 face-down 유지)
- **전략적 가치**:
  - King 위치 파악 → 빈 열 만들기 전략
  - 높은 Rank 카드 위치 → Foundation 우선순위 결정
  - 막힌 상황에서 "다음 수" 계획 수립
- **제약**: 한 번에 1장만 볼 수 있음 (동시 다중 Peek 불가)

### Bomb 듀얼 유즈

Bomb 야생카드는 플레이어가 **어디에 사용하느냐에 따라** 효과가 달라진다.

| 사용 위치 | 효과 | 상황 |
|----------|------|------|
| **Foundation에 드랍** | 전체 화면 폭발 공격 (범위 데미지 + 기절 2초) | 적이 몰려올 때 긴급 방어 |
| **Tableau에 드랍** | face-down 카드 2장 자동 뒤집기 (무작위) | 솔리테어 막힌 상황 돌파 |

- **Foundation 사용 시**:
  - 화면 상단 전투 영역 전체에 폭발 이펙트
  - 모든 적에게 중간 데미지 (일반 투사체의 3배) + 2초 기절
  - 투사체 Tier는 상승하지 않음 (전투 전용 소모품)
  - 음향: 큰 폭발음 + 화면 흔들림
- **Tableau 사용 시**:
  - Tableau 전체에서 face-down 카드 2장을 무작위로 선택하여 자동 뒤집음
  - face-down이 2장 미만이면 있는 만큼만 뒤집음 (0장이면 효과 없음)
  - 시각: 뒤집힌 카드에 붉은 섬광 효과

**트레이드오프 설계**:
- 전투가 위급하면 Foundation에 써서 즉시 위기 탈출
- 솔리테어가 막히면 Tableau에 써서 정보 획득 + 진행 가속
- "지금 쓸까, 나중을 위해 아낄까" 타이밍 결정

### 야생카드 관리 규칙

- **슬롯 시스템**: 화면 하단 중앙에 야생카드 전용 슬롯 (초기 최대 3개)
- **드랍 획득**: 적 처치 시 야생카드가 전투 화면에 짧게 표시된 후 슬롯으로 자동 이동
- **슬롯 초과**: 슬롯이 가득 찬 상태에서 새 야생카드 획득 시 가장 오래된 카드 소멸 (FIFO)
- **사용 방법**: 슬롯에서 드래그하여 Foundation 또는 Tableau로 이동
- **표준 규칙과의 분리**: 야생카드는 솔리테어 표준 이동 규칙(색상 교대, 내림차순 등)과 무관하게 전용 슬롯에서만 관리됨
- **레벨업 확장**: "솔리테어 보조" 레벨업 선택지 중 "야생카드 슬롯 +1" 존재 (최대 5개까지 확장 가능)

## 솔리테어 막힘 안전장치 (4중 방어)

Klondike 솔리테어의 본질적 한계: 무작위 딜에서 약 25%는 완전 풀이 불가능. 이를 보완하기 위한 4단계 안전망.

### 안전장치 철학

- **Klondike 규칙 보존**: 안전장치는 솔리테어 규칙 자체를 변경하지 않음 (Tableau 셔플, 카드 Swap 등 불채택)
- **외부 보조**: 표준 52장 덱 외부에서 도움 제공 (야생카드, 자동 reveal 등)
- **단계적 완화**: 가벼운 안전장치(Stock 무한 순환) → 중간(레벨업 보조) → 강력(야생카드) → 최후(새 덱 딜링)

### 4중 안전장치 체계

| 안전장치 | 트리거 | 효과 | Klondike 규칙 위반? | 효과 범위 |
|---------|--------|------|-------------------|----------|
| **1. Stock 무한 순환** | 항시 | Waste → Stock 무한 반복 가능 | 아니오 (표준 규칙) | 기본 |
| **2. 레벨업 솔리테어 보조** | Wave 3, 6, 9 레벨업 시 | "5장 드로우", "face-down peek", "야생카드 슬롯+1" 선택 가능 | 아니오 (게임 외 보조) | 중간 |
| **3. 웨이브 클리어 reveal** | 웨이브 종료 시 | face-down 1장 자동 공개 | 상태 변경이나 규칙은 미변경 | 경량 |
| **4. 야생카드 (Joker/Peek/Bomb)** | 적 처치 시 15% | Foundation 돌파구 / 정보 제공 / 뒤집기 | 아니오 (외부 카드) | 강력 |

### 안전장치 1: Stock 무한 순환

- **규칙**: Klondike 표준 규칙. Waste 카드를 모두 넘기면 Stock을 다시 채워서 무한 반복 가능
- **게임플레이 효과**:
  - "막힌 것처럼 보이는" 상황에서도 Stock을 계속 넘기며 Waste 카드를 재확인할 기회
  - Stock 순환 자체가 전체 적 감속(0.5초) 효과 → 방어적 가치
- **한계**: Stock/Waste에 유용한 카드가 없으면 무의미한 반복

### 안전장치 2: 레벨업 솔리테어 보조

레벨업 선택지 3카테고리 중 **"솔리테어 보조"** 카테고리가 막힘 완화 전담.

| 레벨업 선택지 | 효과 | 출현 빈도 |
|------------|------|----------|
| **5장 드로우 (Draw Burst)** | 한 번만 사용 가능. Stock에서 5장을 한꺼번에 Waste로 드로우 (3장 대신) | Wave 3, 6, 9 |
| **face-down 미리보기 (Peek All)** | 10초간 모든 face-down 카드를 반투명 오버레이로 표시 (Peek 야생카드의 전체 버전) | Wave 3, 6, 9 |
| **야생카드 슬롯 +1** | 야생카드 최대 보유 수 +1 (초기 3 → 최대 5까지 확장) | Wave 3, 6, 9 |

- **출현 타이밍**: Wave 3, 6, 9 클리어 시 레벨업 발동. 3카테고리(투사체 강화 / 생존 강화 / 솔리테어 보조) 중 1개를 무작위로 3개씩 제시
- **선택 전략**: 솔리테어가 막힌 상태면 "솔리테어 보조" 선택 우선도 상승. 전투가 위급하면 "생존 강화" 우선

### 안전장치 3: 웨이브 클리어 reveal

- **효과**: 웨이브 종료 시 face-down 카드 1장 자동 뒤집기 (무작위)
- **발동 빈도**: 웨이브당 1회 (총 10회/판)
- **누적 효과**: 10웨이브 진행 중 최대 10장의 face-down을 자동으로 공개 → 솔리테어 정보량 증가
- **규칙 보존**: 카드를 뒤집는 행위는 Klondike에서 정상 동작. 자동화했을 뿐 규칙 자체는 변경 없음

### 안전장치 4: 야생카드 시스템

위 "야생카드 시스템" 섹션 참조. 특히:
- **Joker**: 막힌 Foundation을 강제 돌파
- **Peek**: 정보 부족으로 인한 막힘 해소
- **Bomb (Tableau 사용)**: face-down 2장 강제 뒤집기

### 최후 수단: 솔리테어 완전 클리어 후 옵션

- **트리거**: 52장을 모두 Foundation에 올려서 Tableau/Stock/Waste가 완전히 비었을 때
- **효과**: "새 덱 딜링" 버튼 활성화
  - 클릭 시 새로운 52장 덱을 무작위로 딜링하여 솔리테어 리셋
  - Foundation은 **유지** (투사체 Tier 그대로)
  - 야생카드 슬롯도 유지
- **게임플레이 의미**:
  - 솔리테어를 완전히 클리어한 플레이어에게 "무한 투사체 강화" 경로 제공
  - 52장 클리어 = 이미 강력한 상태 → 새 덱으로 더 강해질 수 있음
- **밸런싱**: 10분 내에 52장 클리어는 상당히 어려움 → Wave 8-9쯤 가능. 최종 웨이브에서 압도적 화력 보상

## 메카닉 연결점: 솔리테어 ↔ 서바이버 매핑

양방향 피드백 루프의 전체 연결 구조를 요약한 테이블.

### 순방향 연결 (솔리테어 → 전투)

| 솔리테어 개념 | 서바이버 개념 | 연결 방식 | 분류 |
|-------------|-------------|----------|------|
| Foundation 적재 (카드 올리기) | 투사체 강화 (Tier 상승 + 즉발 + 넉백) | 카드를 올릴 때마다 즉시 전투 파워 증가 | 액티브 |
| Tableau → Tableau 이동 | 체인 게이지 충전 → 발사속도 버스트 | 연속 이동으로 게이지 차면 3초간 발사속도 2배 | 액티브 |
| face-down 뒤집기 | 경험치 획득 (5 XP) | Tableau 정리 과정에서 레벨업 진행 | 패시브 |
| Stock 순환 (3장 드로우) | 전체 적 감속 (0.5초, -30%) | Stock 클릭 시 자동 발동 | 패시브 |
| 빈 Tableau 열 생성 | 결계 존 (해당 영역 적 감속 50%) | 빈 열에 대응하는 상단 영역에 감속 존 생성 | 패시브 |

### 역방향 연결 (전투 → 솔리테어)

| 서바이버 개념 | 솔리테어 개념 | 연결 방식 | 긍정/부정 |
|-------------|-------------|----------|----------|
| 웨이브 클리어 (생존) | face-down 1장 자동 reveal | 웨이브 종료 시 무작위 1장 뒤집기 | 긍정 (보상) |
| 적 처치 (전투 성과) | 야생카드 드랍 (15% 확률) | Joker/Peek/Bomb 획득으로 돌파구 제공 | 긍정 (보상) |
| 미니보스 처치 (보스전 승리) | 확정 야생카드 1장 | Wave 3, 6, 9 보스 처치 시 100% 드랍 | 긍정 (보상) |
| 피격 / 보스 등장 | ~~카드 조작 불가 / 저주~~ | **불채택** (하향 나선 방지) | 부정 (X) |

**설계 원칙**: 역방향은 **긍정적 보상만** 존재. 전투 실패가 솔리테어를 더 어렵게 만드는 패널티는 일체 없음.

### 야생카드 ↔ 솔리테어 규칙

| 야생카드 타입 | Klondike 규칙 관점 | 효과 | 규칙 위반 여부 |
|-------------|------------------|------|--------------|
| Joker | "와일드 카드" (덱 외부) | 아무 Foundation에 다음 Rank로 올림 | 아니오 (외부 카드) |
| Peek | 정보 제공 (덱 변경 없음) | face-down 카드 내용 10초간 표시 | 아니오 (정보만) |
| Bomb (Tableau 사용) | 자동 뒤집기 (덱 변경 없음) | face-down 2장을 자동으로 reveal | 아니오 (뒤집기는 정상 동작) |
| Bomb (Foundation 사용) | 전투 전용 (솔리테어 무관) | 전체 화면 폭발 공격 | 해당 없음 |

**규칙 보존 원칙**: 야생카드는 모두 "표준 52장 덱 외부"의 치트 아이템이지만, 사용 시 Klondike의 **핵심 규칙**(색상 교대, 내림차순, A→K 순서 등)은 변경하지 않는다.

## 안전장치 효과 예측

4중 안전장치가 결합되어 "풀이 불가능한 덱"에서도 진행 가능성을 얼마나 보장하는가?

### 안전장치 누적 효과 시뮬레이션

| 상황 | 안전장치 없음 | Stock 무한 순환 | + 야생카드 | + 웨이브 reveal | + 레벨업 보조 |
|------|-------------|---------------|----------|--------------|-------------|
| 운 좋은 덱 (상위 25%) | 100% 진행 | 100% | 100% | 100% | 100% |
| 평범한 덱 (중간 50%) | 60% 진행 | 75% | 90% | 95% | 98% |
| 운 나쁜 덱 (하위 25%) | 0-20% 진행 | 30% | 60% | 75% | 85% |

- **Stock 무한 순환**: 기본 안전망. 반복해서 Waste 확인 가능 → 약 15% 진행률 향상
- **야생카드**: Joker(7%) + Peek(5%) + Bomb(3%) = 15% 드랍. 웨이브당 평균 5-8장 획득 → 막힌 Foundation/Tableau 돌파 → 약 30% 진행률 향상
- **웨이브 reveal**: 10웨이브 × 1장 = 10장 자동 뒤집기 → 정보량 증가 → 약 15% 진행률 향상
- **레벨업 보조**: Peek All, 5장 드로우, 슬롯+1 등 선택적 사용 → 약 10% 진행률 향상

**결론**: 4중 안전장치 결합 시 "운 나쁜 덱"에서도 85% 이상 진행 가능 → 플레이어가 "운 때문에 막혔다"는 느낌 대신 "야생카드를 잘 쓰면 돌파할 수 있다"는 에이전시(agency) 보유.

### 안전장치 발동 타임라인 (전형적 10분 세션)

```
00:00 - Game Start, 52장 딜링
00:30 - Wave 1 전투
01:00 - Stock 순환 시작 (안전장치 1 활성)
02:00 - 첫 야생카드 획득 (Peek) → 사용 여부 판단
03:00 - Wave 3 클리어 → 레벨업 "야생카드 슬롯 +1" 선택 (안전장치 2)
03:30 - Wave 3 종료 → face-down 1장 자동 reveal (안전장치 3)
04:00 - 야생카드 Joker 획득 → Heart Foundation에 사용 (막힌 상태 돌파)
06:00 - Wave 6 클리어 → 레벨업 "Peek All" 선택 → 전체 face-down 확인
07:00 - Bomb 야생카드 2개 보유 → Tableau에 사용하여 face-down 2장 뒤집기
09:00 - Wave 9 클리어 → 레벨업 "5장 드로우" 선택 → Stock 5장 한 번에 확인
10:00 - Wave 10 최종 보스전
```

**관찰**: 중간중간 "막힌 것 같은" 순간마다 안전장치가 발동하여 돌파구 제공 → 플레이어는 지속적인 진행감 유지.

## 구현 우선순위

피드백 루프와 안전장치 시스템의 구현 순서 제안.

### Phase 1: 순방향 기본 (MVP)

1. Foundation 적재 → 투사체 Tier 상승 + 즉발 1발
2. face-down 뒤집기 → 경험치 5 XP
3. Stock 무한 순환 (표준 Klondike 규칙)

**목표**: 솔리테어→전투의 기본 연결 확인. 플레이 가능한 루프 구축.

### Phase 2: 안전장치 핵심

4. 야생카드 시스템 (Joker만 우선, Peek/Bomb은 Phase 3)
5. 웨이브 클리어 → face-down 1장 자동 reveal
6. 레벨업 "솔리테어 보조" 선택지 (5장 드로우만 우선)

**목표**: 솔리테어 막힘 완화. 10분 세션 완주 가능성 검증.

### Phase 3: 전술적 깊이

7. 체인 게이지 시스템 (Tableau 이동 → 발사속도 버스트)
8. 빈 Tableau 열 → 결계 존
9. Stock 순환 → 전체 적 감속 0.5초
10. 야생카드 Peek, Bomb (듀얼 유즈)
11. Foundation 적재 → 넉백 효과

**목표**: 전략적 선택 확대. "언제 무엇을 쓸까" 타이밍 게임.

### Phase 4: 폴리싱

12. 모든 시각/음향 피드백 강화
13. 레벨업 "솔리테어 보조" 전체 선택지 (Peek All, 슬롯+1)
14. 솔리테어 완전 클리어 후 새 덱 딜링 옵션
15. 야생카드 슬롯 UI 애니메이션

**목표**: 느낌(juice) 극대화. 모든 행동에 즉각적 피드백.

## 밸런싱 레버

피드백 루프와 안전장치의 주요 밸런싱 변수.

| 시스템 | 변수 | 초기값 | 조정 방향 |
|--------|------|--------|----------|
| 체인 게이지 | 충전량 (Tableau 이동 1회) | +10% | 너무 느리면 +15%, 너무 빠르면 +7% |
| 체인 게이지 | 콤보 윈도우 | 10초 | 너무 빡빡하면 12초로 완화 |
| 체인 브레이크 | 지속 시간 | 3초 | 너무 강하면 2초, 너무 약하면 4초 |
| 야생카드 | Joker 드랍률 | 7% | 너무 쉬우면 5%, 너무 어려우면 10% |
| 야생카드 | Peek 드랍률 | 5% | 정보 부족 시 7%로 상향 |
| 야생카드 | Bomb 드랍률 | 3% | 폭발 사용 빈도 보고 조정 |
| 야생카드 슬롯 | 초기 슬롯 수 | 3개 | 너무 적으면 4개로 상향 |
| Bomb (Foundation) | 폭발 데미지 | 일반 투사체 x3 | 너무 강하면 x2.5, 너무 약하면 x4 |
| Bomb (Tableau) | 뒤집기 개수 | 2장 | 너무 약하면 3장 |
| 결계 존 | 감속 비율 | 50% | 너무 강하면 40%, 너무 약하면 60% |
| Stock 순환 | 감속 지속 시간 | 0.5초 | 너무 짧으면 0.8초 |
| 웨이브 reveal | 자동 뒤집기 개수 | 1장 | 너무 적으면 2장 (단, 너무 쉬워질 위험) |
| 레벨업 5장 드로우 | 드로우 개수 | 5장 | 너무 많으면 4장, 너무 적으면 6장 |

**밸런싱 목표**:
- 안전장치가 너무 강하면 솔리테어 긴장감 소멸 → 드랍률/효과 하향
- 안전장치가 너무 약하면 막힘 좌절 → 드랍률/효과 상향
- 목표 완주율: 85-90% (상위 10-15%만 막힘 경험)

## 요약

"Solitaire Survivors"의 피드백 시스템은 **양방향 긍정적 나선**을 핵심으로 한다. 솔리테어를 잘 풀수록 전투가 강해지고, 전투를 잘할수록 솔리테어 돌파구가 생긴다. 부정적 역방향(피격 패널티, 보스 저주 등)은 일체 배제하여 "약해질수록 더 약해지는" 하향 나선을 방지한다.

4중 안전장치(Stock 무한 순환, 레벨업 보조, 웨이브 reveal, 야생카드)는 Klondike의 본질적 막힘 문제를 완화하되 **규칙 자체는 변경하지 않는다**. 모든 안전장치는 "표준 52장 덱 외부"에서 보조하며, 플레이어에게 "운이 아닌 선택으로 돌파할 수 있다"는 에이전시를 부여한다.

순방향 피드백은 **액티브 2개 (Foundation 적재, 체인 게이지) + 패시브 3개**로 구성하여 인지 부하를 제한하면서도 전략적 깊이를 제공한다. 야생카드의 듀얼 유즈(특히 Bomb)는 "공격 vs 솔리테어 진행"이라는 의미 있는 트레이드오프를 만든다.

최종 목표: **10분 내내 "다음 수가 있다"는 느낌을 유지하며, 85-90% 완주율을 달성**하는 것.
