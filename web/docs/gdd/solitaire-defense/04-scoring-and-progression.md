# Scoring & Progression - 스코어링 및 진행 시스템

## 스코어링 시스템

### 점수 구성 요소

게임 종료 시 (승리 또는 Game Over) 최종 점수가 산출된다.

| 항목 | 점수 | 설명 |
|------|------|------|
| 웨이브 클리어 | 웨이브 번호 x 100 | Wave 1 = 100, Wave 10 = 1000 |
| 적 처치 | 적 1체당 10 | 보스: 50 |
| Foundation 카드 | 카드 1장당 20 | 52장 전부 = 1040 |
| 기지 잔여 HP | HP 1당 2 | 최대 400 (200 HP x 2) |
| Suit 콤보 보너스 | 콤보당 25 | 콤보 발동 횟수 누적 |
| 마일스톤 보너스 | 마일스톤당 50 | 달성한 마일스톤 수 |
| 챔피언 소환 | 챔피언 1체당 200 | 최대 4체 = 800 |
| 턴 효율 | 미사용 턴 x 5 | 웨이브당 남은 턴 합산 |
| Perfect Wave | 500 | 기지 피해 없이 웨이브 클리어 |

### 등급 시스템

| 등급 | 점수 범위 | 조건 |
|------|----------|------|
| S | 6000+ | Wave 10 클리어 + 기지 HP 50% 이상 |
| A | 4500-5999 | Wave 10 클리어 |
| B | 3000-4499 | Wave 7 이상 도달 |
| C | 1500-2999 | Wave 4 이상 도달 |
| D | 0-1499 | Wave 4 미만 |

### 리더보드

- **시드 기반 랭킹**: 같은 시드(같은 카드 배치)에서 다른 플레이어와 점수 비교
- 기존 Solitary의 seeded shuffle (Mulberry32 PRNG) 활용
- 일간/주간/전체 리더보드

## 난이도 곡선

### 설계 원칙

1. **초반 (Wave 1-3)**: 학습 구간. 솔리테어와 디펜스 각각의 기본 메카닉 습득
2. **중반 (Wave 4-7)**: 전략 구간. Suit 속성, 콤보, 마일스톤 활용 필요
3. **후반 (Wave 8-10)**: 마스터 구간. 최적 플레이 없이는 클리어 불가

### 초반 (Wave 1-3): 학습

- Wave 1은 튜토리얼 성격 (턴 20, 적 5마리, Grunt만)
- 이 구간에서 플레이어가 학습해야 할 것:
  - Card Phase에서 Foundation에 카드 올리기 = 유닛 생산
  - Deploy Phase에서 유닛 배치
  - Battle Phase 관전
- **목표**: Bronze 유닛 3-4체로 충분히 클리어 가능

### 중반 (Wave 4-7): 전략

- 적 구성이 다양해지고, 보스 등장
- Suit 속성 웨이브 (Wave 5) 시작
- **목표**: Silver 유닛 활용, Suit 콤보, 유닛 배치 최적화 필요
- 챔피언 유닛 1체 소환 가능하면 유리

### 후반 (Wave 8-10): 마스터

- 턴 제한 축소 (12턴)
- 복합 속성 내성 (Wave 8)
- 최종 보스 3페이즈
- **목표**: Gold 유닛 + 챔피언 유닛 조합, 정밀한 Tableau 관리 필수

### 난이도 밸런스 지표

| 지표 | 목표 수치 |
|------|----------|
| Wave 1 클리어율 | 95% |
| Wave 5 도달율 | 70% |
| Wave 7 도달율 | 50% |
| Wave 10 클리어율 | 25% |
| S 등급 달성율 | 5% |

## 게임 모드 변형

### Normal Mode (기본)
- 위에 서술한 표준 10 웨이브
- 기지 HP 200, 표준 턴 제한

### Endless Mode (무한 모드)
- Wave 10 이후에도 계속 진행
- 적 HP/ATK가 웨이브당 x1.2씩 증가
- 더 이상 새로운 카드가 없으므로 (52장 한정) 기존 유닛으로 버텨야 함
- 최고 웨이브 기록이 점수

### Quick Mode (빠른 모드)
- 5 웨이브 구성 (1, 3, 5, 7, 10에 해당하는 난이도)
- 턴 제한 +5 (여유로운 플레이)
- 캐주얼 플레이용

## 리플레이 요소

1. **시드 선택**: 다른 시드 = 다른 카드 배치 = 다른 전략 필요
2. **전략 분기**: 같은 시드에서도 Suit 우선순위에 따라 다른 결과
3. **등급 도전**: S 등급 달성을 위한 최적화 플레이
4. **무한 모드**: 최고 웨이브 기록 갱신
5. **속성 웨이브 대응**: 랜덤 Suit 내성에 따른 즉흥 전략 변화
